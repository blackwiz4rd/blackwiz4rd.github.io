## üí≥ RCH Pay SoftPOS

**RCH Pay SoftPOS** is a mobile-first payment application that transforms any compatible Android device into a **secure, certified payment terminal**.  

### Key Features
- Accepts **contactless payments** from credit/debit cards and digital wallets (Google Pay, Apple Pay, etc.).  
- Integrates with **RCH‚Äôs fiscal systems** for seamless transaction tracking and reporting.  
- Fully compliant with **PCI DSS** and **EMV standards**.  
- Provides a **user-friendly onboarding flow** for merchants.  

### Technology Highlights
- Developed in **Kotlin**, optimized for Android devices.  
- Built using **Koin** for dependency injection.  
- Integrates **Firebase Performance SDK** for real-time monitoring.  
- Features advanced **state management**, **transactional flow logic**, and **error handling** systems to ensure reliability.
